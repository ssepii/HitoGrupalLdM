Orígenes y uso

JSON(JavaScript Object Notation)


JSON fue creado en 2002 por Douglas Crockford como una alternativa más fácil de usar comparado a XML. JSON utiliza un formato de texto plano basado en pares clave-valor para describir datos, lo que hace más fácil el trabajo para los desarrolladores de aplicaciones web. La sintaxis de JSON es más sencilla que la de XML, lo que la hace ideal para su uso en aplicaciones web.

Con AJAX también se pueden recibir datos en formato JSON desde un servidor y acceder a ellos mediante JavaScript para mostrarlos en la página HTML.

Como ya se ha mencionado antes, JSON es más fácil de manejar que XML, ya que utiliza un formato más legible y sencillo. Por ejemplo, los datos se reciben en formato JSON desde un servidor y luego se accede a ellos mediante JavaScript para mostrarlos en la página HTML.
